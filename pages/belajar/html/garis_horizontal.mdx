# GARIS HORIZONTAL HTML
## Apa itu garis horizontal 

HTML elemen tag `<hr>` *(horizontal rule)* merupakan elemen pada HTML yang digunakan untuk membuat garis datar memanjang secara horizontal.
Elemen `<hr>` sering digunakan saat terjadi perubahan tema bahasan pada level 
elemen paragraf sehingga diperlukan sebuah garis horizontal sebagai pemisah dengan bagian di bawahnya.

### Mengenal Tag HR

Kepanjangan dari HR adalah *horizontal rule*, yaitu sebuah garis lurus yang terbentuk secara horizontal,
tag ini termasuk single tag, atau tag tunggal, dimana tidak ada tag penutup setelahnya, 
berikut ini beberapa atribut tag `<hr>` yang bisa di gunakan.

|Atribut	| Nilai               |	Keterangan|
| :---:   | :---:               | :---:     |
| align   |	left, center, right |	Untuk menentukan posisi garis.  |
| size	  | pixels              |	Untuk menentukan ukuran garis.  |
| width   |	pixels, %           |	Untuk menentukan lebar garis.   |

## Cara Membuat Garis Horizontal

Contoh berikut ini adalah cara sederhana untuk membuat garis horizontal tanpa, 
silahkan buka text editor anda lalu ketikan kode di bawah ini.
```
<body>
  membuat garis horizontal<hr/>
  
  <P>
    lorem ipsum dolor sit amet,ceknsectetur adipisicing elit,
    <br/>
    sed do eiusmod tempor incididunt ut labore et dolore magna
aliqua.
<br/>
  ut enim ad minim veniam,quis nostrud exercitation ullamco
laboris.
  nisi ut aliquip ex ea commodo consequat.
<hr/>

 </body>
 ```
