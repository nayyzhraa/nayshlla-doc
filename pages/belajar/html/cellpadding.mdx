#  CELLPADDING HTML 
## Apa itu Celllpadding? 

  Cellpadding adalah atribut dari sel individu dalam tabel, sehingga setiap sel dalam tabel dapat memiliki nilai cellpadding sendiri. 
Atribut cellpadding ditambahkan ke bahasa HTML versi 2.0 pada tahun 1996. Ruang antara teks dan batas merupakan elemen penting dari desain halaman web, 
karena meningkatkan keterbacaan teks dan daya tarik visual grafik dalam sel tabel.

### Contoh kode dari cellpadding html:
```
<table border="4">
<tr>
<th>No</th>
<th>Nim</th>
<th>Nama</th>
</tr>
<tr>
<td>1</td>
<td>2018089</td>
<td>Nayshilla zahra</td>
</tr>
<tr>
<td>2</td>
<td>2080396</td>
<td>nayzahra</td>
</tr>
<tr>
<td>3</td>
<td>2080356</td>
<td>nyshllazhr</td>
</tr>
</table>
</body>
</html>
```
