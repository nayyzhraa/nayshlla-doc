# HTML Headings
# Apa Itu HTML Headings ?
HTML Headings adalah judul/sub-judul yang ingin ditampilkan dalam 
halaman website HTML

HTML Headings didefinisikan dengan `<h1>` hingga `<h6>`, contohnya:
#### Dalam Bentuk Tulisan
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

#### Dalam Bentuk Code
```
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
  ```
